## [0.1.0] - 2025-06-16

### Added
- Initial Tokyo Night theme implementation (`tokyonight.sh`)
- TPM plugin loader (`tokyo-night-tmux.tmux`) 
- Support for tmux plugin manager (tpm) installation
- Modern dark theme based on Tokyo Night color palette
- Basic theme configuration and setup

### Installation
```bash
# Via tpm
set -g @plugin "joaofelipegalvao/tokyo-night-tmux"
```
